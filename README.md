# Automatización de Excel con Python

Este proyecto lee un archivo Excel y envía emails automáticamente según condiciones.

## Tecnologías
- Python
- openpyxl
- smtplib

## Funcionalidad
- Detecta ventas mayores a 3000
- Envía emails automáticamente

## Cómo usar
1. Colocar archivo ventas.xlsx
2. Ejecutar: python codigo.py
